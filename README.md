# MLBAM Challenge



## How to get started:

- [ ] clone or fork the repo
- [ ] cd (change directory) into the correct folder
- [ ] `npm install`
- [ ] In the terminal type `npm start server.js`
- [ ] In your browser, navigate to `localhost:3000`
